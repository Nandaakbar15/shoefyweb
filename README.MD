# Website Toko Sepatu Shoefy
**Sebuah website toko sepatu sederhana 🏪**

#### Main Feature 🛠️
- Admin : CRUD Data Produk
- Customer : Lihat list sepatu dan detail sepatu

#### Tech Stack 💻
- Frontend : Vue.js + Bootstrap
- Backend : Express.Js
- Database : MySQL
